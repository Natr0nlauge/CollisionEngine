Detailed Description

Circular RigidBody.

This is displayed as a Polygon with a lot of corners so it looks "smooth". To avoid iterating through a lot of corners and edges, Circle is a seperate class. Its only geometrical parameter is m_radius (m_resolution only influences appearance but not behaviour).
