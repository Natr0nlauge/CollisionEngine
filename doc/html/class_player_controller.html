Detailed Description

Enables the bodies in the simulation to be controlled by the user.

This class holds a pointer to the RigidBody that is supposed to be controlled by the player. The same pointer is not included in Simulation::m_collisionPartners. Also holds settings for controlling the body (acceleration, buttons for controls).
